# Scripting
Scripts for NextFEM Designer scripting engine

Scripting engine can be found in NextFEM Designer by clicking on the "Verification" button (Results menu) or by pressing CTRL+G.

![alt text](https://www.nextfem.it/api/scriptingEngine.JPG)

<div class="WordSection1"><h2><a name="_Toc72938271"><span lang="EN-GB" xml:lang="EN-GB">Custom verifications</span></a></h2>
<p class="MsoNormal" style="text-align:justify"><span lang="EN-GB" xml:lang="EN-GB">It is possible to customize the list of verifications with the notation described in the following. The text file containing checking can be placed in the “<i>verification</i>” folder in the program installation directory and must have the “.<i>nvv</i>” extension (see for example <i>trusses.nvv</i>). Such file can be edited with Notepad++ (<span class="MsoHyperlink"><a href="https://notepad-plus-plus.org/">https://notepad-plus-plus.org/</a></span>). A setting for the syntax highlight for Notepad++ can be found in the “<i>verification</i>” folder (<i>NextFEMVerifications.xml</i>).</span></p>
<p class="MsoNormal" style="text-align:justify"><span lang="EN-GB" xml:lang="EN-GB">The checking engine supports the code execution in blocks. Each block is delimited by the identifiers as in the following example and must the named with a floating point number (0.6).</span></p>
<p style="font-style: italic; font-weight: bold" class="MsoNormal"><span lang="EN-GB" xml:lang="EN-GB">$$0.6</span></p>
<p style="font-style: italic" class="MsoNormal"><span lang="EN-GB" xml:lang="EN-GB"># this is a comment</span></p>
<p style="font-style: italic" class="MsoNormal"><span lang="EN-GB" xml:lang="EN-GB">execif(SecType==1,1.0)</span></p>
<p style="font-style: italic; font-weight: bold" class="MsoNormal"><span lang="EN-GB" xml:lang="EN-GB">$!</span></p>
<p class="MsoNormal" style="text-align:justify"><span lang="EN-GB" xml:lang="EN-GB">To call each block, the following keywords are available:</span></p>
<p class="MsoListParagraphCxSpFirst" style="text-indent:-18.0pt"><span lang="EN-GB" style="font-family:&quot;Arial&quot;,sans-serif" xml:lang="EN-GB">-<span style="font:7.0pt &quot;Times New Roman&quot;">      </span></span> <i><span lang="EN-GB" xml:lang="EN-GB">exec(0.6)</span></i> <span lang="EN-GB" xml:lang="EN-GB">: executes the code block named 0.6</span></p>
<p class="MsoListParagraphCxSpMiddle" style="text-indent:-18.0pt"><span lang="EN-GB" style="font-family:&quot;Arial&quot;,sans-serif" xml:lang="EN-GB">-<span style="font:7.0pt &quot;Times New Roman&quot;">      </span></span> <i><span lang="EN-GB" xml:lang="EN-GB">execif(condition, 0.6)</span></i><span lang="EN-GB" xml:lang="EN-GB">: executes the code block named 0.6 is <i>condition</i> is true;</span></p>
<p class="MsoListParagraphCxSpLast" style="text-indent:-18.0pt"><span lang="EN-GB" style="font-family:&quot;Arial&quot;,sans-serif" xml:lang="EN-GB">-<span style="font:7.0pt &quot;Times New Roman&quot;">      </span></span> <i><span lang="EN-GB" xml:lang="EN-GB">execwhile(condition, 0</span></i><span lang="EN-GB" xml:lang="EN-GB">.6): executes the code block named 0.6 while condition is true. The variable <i>exitdo</i> equal to 1 within the code block forces the exit. Limited to 10000 cycles.</span></p>

<p class="MsoNormal" style="margin-bottom:0cm;margin-bottom:.0001pt;text-indent: 35.4pt"><span lang="EN-GB" xml:lang="EN-GB">Equal: <b>==</b></span></p>

<p class="MsoNormal" style="margin-bottom:0cm;margin-bottom:.0001pt;text-indent: 35.4pt"><span lang="EN-GB" xml:lang="EN-GB">Arc cotangent: <b>acot</b></span></p>
